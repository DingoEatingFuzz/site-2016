Editor's note: most of my work has been on B2B tools that are hidden behind auth walls. I'm happy to talk about them, but I can't flaunt them here.

  • 2016 Emoji Skin Tone Randomizer

    • JavaScript
    • Chrome

    A chrome extension for assigning a random skin tone to a skin tone eligible emoji that doesn't already have one assigned.

  • 2015 Headers Middleman

    • JavaScript
    • Chrome

    A chrome extension for modifying the headers of HTTP Requests based on regex pattern matching.

  • 2015 Climb Tracker

    • JavaScript
    • React

    A simple web app for tracking bouldering workouts. It focuses on quickly marking which problems were completed and which were attempted. In this way, at the end of the workout, a histogram and tally of problems and problem difficulties is plotted.

  • 2015 NPR Music We Love Bookmarklets

    • JavaScript
    • Browsers

    The NPR Music We Love app was a wonderful things, but it had no way to pin/favorite/star/save the tracks you liked. This was solvable in many ways, but the way that sounded the most interesting at the time was bookmarklets. With a click of a bookmarklet, the track would be saved to localstorage. I went overboard and created additional bookmarklets for seeing the track list and for disliking tracks. You know, just for fun.

  • 2014 Github Avatar Arrangement

    • Python
    • Processing

    A quick sketch that generates all possible Github Avatars (not including color variations). Since Github default avatars are created through the toggling of 15 states, the resulting space is only 215. Which is high, but not so high it isn't presentable in a single image.
